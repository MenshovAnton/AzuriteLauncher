# CubeX Launcher
Minimalistic Minecraft launcher with a classic UI and modern features.

## Tech Stack
- Rust (Tauri)
- React + TypeScript

## Planned features
- Minimalistic nostalgia UI (from time Minecraft 1.10)
- Mod/Modpacks Manager
- Game instances (Containers)
- Basic security checks

## Status
In Development. Early Stage.

## Run
```
pnpm install 
pnpm tauri dev
```
